# Aeronomy Payout Platform Frontend

This project is a frontend informational website created for Aeronomy Technologies Private Limited as part of a technical evaluation task.

The website represents a conceptual digital payout infrastructure platform used by modern businesses to manage automated financial workflows and digital transactions.

## Features

- Modern fintech style user interface
- Animated background network effect
- Informational sections explaining payout infrastructure
- Platform capability cards
- Responsive layout
- Scroll animation effects
- Single page frontend design

## Technologies Used

HTML  
CSS  
JavaScript

## Project Type

Frontend only informational website.

No SEO optimization included as per project requirements.
